# 📖 Manual de Usuario - Marketplace Moda Circular Luxury

Bienvenido al manual oficial de la plataforma. Este documento detalla cómo utilizar las funciones clave del sistema para compradores y vendedores.

## 🛍️ Para Compradores
1. **Búsqueda**: Utiliza el grid de búsqueda para filtrar prendas por categoría y estado.
2. **Compra**: Al confirmar un pedido, recibirás los datos de contacto del vendedor (WhatsApp) para coordinar la entrega.
3. **Estado**: Las prendas marcadas como "Vendido" no pueden ser añadidas al carrito.

## 👗 Para Vendedores (Modo Business)

### 1. Registro Gratuito y Libre
La plataforma permite el registro gratuito para todos los vendedores. No existen mensualidades ni comisiones por venta dentro del sistema. Al registrarte, recibes **2 créditos de IA** de regalo.

### 2. Publicación Ilimitada
Puedes subir tantas prendas como desees a tu closet virtual. No existen límites de inventario para ningún vendedor.

### 3. Publicación con IA (Magic Listing)
Al subir una foto de tu prenda, puedes utilizar créditos de IA para generar automáticamente el título, descripción y etiquetas sugeridas. Una vez agotados los créditos iniciales, puedes adquirir paquetes adicionales en tu panel.

### 4. Gestión de Inventario
Desde tu perfil puedes ver tus prendas activas, marcarlas como "Vendido" (para que el comprador sepa que ya no está disponible) o eliminarlas.

## 👔 Panel de Administrador (Fundadores)
Los usuarios con rango de Administrador disfrutan de:
- **Estatus Ilimitado**: Publicación de prendas sin límite.
- **Créditos IA Infinitos**: Uso libre del motor de descripciones automáticas.
- **Acceso a Reportes**: Visualización de contactos de compradores para soporte.
